# ElectroMart Frontend

This is the frontend application for ElectroMart, an AI-powered product inquiry and sales assistant.

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS

## Features
- Chat-based user interface
- AI-powered responses for product inquiries

## Setup

Install dependencies:
npm install

Run the development server:
npm run dev

The app will run on:
http://localhost:3000

## Environment Variables

Create a .env.local file and add:
NEXT_PUBLIC_API_BASE_URL=http://localhost:8001

## Notes
- Backend must be running for full functionality
- Frontend handles UI only; all business logic is in the backend
